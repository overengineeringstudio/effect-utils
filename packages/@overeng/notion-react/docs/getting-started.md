# Getting Started

Install, minimal example, and first render. Aimed at a reader who has never
used `@overeng/notion-react` before but is comfortable with React. Covers:
install + peer deps, the smallest possible `renderPage` example, where to
plug in a Notion client, and what "done" looks like (a rendered page with
default styles). Links out to `concepts/` for anything non-obvious.
